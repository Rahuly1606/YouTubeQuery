# YoutubeQuery

A React application built with TypeScript, Vite, and Tailwind CSS to search and query YouTube content.

## Features

- Modern UI with particle background
- Search functionality for YouTube content
- Responsive design

## Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [Bun](https://bun.sh/) package manager

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Rahuly1606/YouTubeQuery.git
   cd YouTubeQuery
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:

```bash
npm run dev
```


## Project Structure

- `/src` - Main source code
  - `/components` - Reusable UI components
  - `/hooks` - Custom React hooks
  - `/lib` - Utility functions
  - `/pages` - Application pages
